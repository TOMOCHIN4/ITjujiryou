# 制作メモ — かたおか歯科クリニック 採用コピー初稿

## ヒーロー設計の根拠

訴求軸ⓐ「リコール 60 分で患者と深く関わる」を学生のことばに翻訳するキーが
**「私の患者さん」と言える** という所有・担当感だと判断した。

口腔保健科 3-4 年生が実習中に感じる典型的なモヤモヤ:
- ベルトコンベア式のクリーニングへの疑問
- 「担当制で長く見たい」という素朴な憧れ
- 「歯石取り屋」ではない仕事像を求めている

ここに刺さる言葉として「ひとりの患者さんと、60 分」「私の患者さん」を据えた。
詩情に逃げず、体験ベースの具体表現になっている (success_criteria 2 を満たす狙い)。

## 訴求軸ⓐ→ⓑ→ⓒ→ⓓ の順位反映

- **LP**: セクション 2 (ⓐ) → 3 (ⓑ) → 4 (ⓒ) → 5 (ⓓ) で順次。
- **パンフ**: P2 上段ⓐ / 下段ⓑ、P3 上段ⓒ / 中段ⓓ で順位を物理配置で表現。
- **ポスター**: メイン = ⓐ、サブの 1 行に ⓐⓑ 同居、ボディに ⓒ 系 (転居サポート)。

順位崩れ防止のため、デザイナー (トシ) には P2 上段 = ⓐ を最大級扱いするよう申し送り推奨。

## 院長の悔いの「滲ませ方」

要件: 開業 2 年目の若手離職を直接書かない、しかし読後感に残す。

- LP セクション 4 マイクロコピー「院長は『長く居てくれる人』のための環境づくりに、開業以来こだわっています」
- パンフ P3「長く居てくれる人を、本気で待っています」

「本気で」「以来こだわってきた」という時間軸の重みで、過去の事象を直接語らず姿勢として残した。

## 求人法令 機械チェック

避けた表現:
- 年齢限定 (「若い」「20代」など) → 「新卒」のみ使用 (雇用区分なので可)
- 性別限定 (「ママさん」「女性歓迎」など) → 「ライフステージ」「家族の都合」に置換
- 容姿・国籍限定 → 該当なし
- 「明るく挨拶できる人」(院長理想像) → 本文に組み込まず、求めるトーン全体に分散

最終的に社労士確認が必要 (final_plan の方針どおり申し送り)。

## 大阪弁混入チェック

院長の口調を引きずらず、全コピー標準語で統一。「〜やで」「〜さかい」等の方言形・大阪弁特有の終助詞は使っていない。

## 確定情報待ちプレースホルダー

以下は院長確定情報待ちで `[ ※確定後挿入]` 表記:
- 応募フォーム URL
- クリニック電話番号
- クリニック代表メールアドレス
- クリニック住所 (正式表記)

ユウコ経由で 6/15 初稿レビュー前までに院長から取得すれば、確定原稿に差し替え可。

## エンジニア (センシロウ) ノードへの申し送り

LP ワイヤー (N-B) との整合のため、以下を 5/30 までに先出ししたい:
- LP セクション数: **6 + 応募 CTA セクション = 計 7 ブロック**
- 応募フォーム想定項目:
  - 氏名 (必須)
  - 大学名・学年 (必須)
  - 連絡先 (電話 or メール、いずれか必須)
  - ご希望の連絡時間帯 (任意)
  - 見学希望日 (任意)
  - ご質問・ご相談 (任意・自由記述)
- フォーム送信後ページに「応募内容は院長と採用担当のみが確認します」の念押し文を再掲したい

## デザイナー (トシ) ノードへの申し送り

- ヒーロー周辺の視覚要素: 院長または衛生士が患者さんと向き合っている写真 (60 分の意味を視覚化)
- パンフ P2 上段 (ⓐ) の余白を最大にとり、見開き左ページの主役にする
- ポスター メインキャッチは縦組 2 行を想定して書いたが、横組 1 行案も版面次第で検討余地あり

## 迷った点

- ポスターのメインキャッチで「60 分、ひとり。」まで削るかどうか
  → 3 秒勝負としては最短だが、「私の患者さん」の固有感が消えるため不採用
  → 採用した「『私の患者さん』と言える 60 分を。」は 14 文字、視認 OK 範囲と判断
- LP セクション 4 で待遇数字を全部出すか分散するか
  → セクション 4 で集約。パンフ P3 右下ボックスにも掲載で二重露出。LP では「ライフ両立」文脈に数字を埋めて文脈付きで読ませる狙い
