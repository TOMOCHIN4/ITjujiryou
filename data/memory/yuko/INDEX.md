# yuko 個人記憶 — INDEX

ユウコ (CEO秘書・実質COO) の個人記憶インデックス。

## 配下トピック

- `client_handling/` — クライアント別の対応履歴・癖・要注意事項
- `persona_translation/` — 社内ペルソナ → クライアント語への翻訳辞書
- `routing_decisions/` — どの案件をどの兄弟に振ったかの判断記録
- `_scratch/{case_id}/` — 案件中の作業メモ (deliver 後に整理される)
- `_proposals/{case_id}.md` — 会社記憶昇格候補 (サザン儀礼承認待ち)

## アクセス

- 書き: ユウコ本人 (`Write(data/memory/yuko/**)` が settings.json に追加されている)
- 読み: ユウコ本人 (ユウコは全 memory 閲覧可なので結果的に他人記憶も読めるが、自身の記憶は当然本人だけが書く)
- 兄弟 3 名・サザンは Read 不可 (settings.json の deny で物理ガード)
